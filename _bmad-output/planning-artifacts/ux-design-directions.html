Tend — Design Directions

Four layout approaches for the daily planning view. All use the same visual foundation: warm blush palette, Plus Jakarta Sans, warm cream background. Explore each to see which layout feels most like Tend.

A — Airy Column

Centered, generous whitespace. Capacity bar and goals at top, todos below. Closest to Calm's breathing-room philosophy. Feels like a personal planning journal.

Direction A key traits

  • Centered narrow column (max 600px) — focused, not spread out
  • Goals shown as small pills beneath capacity — always visible but quiet
  • Todos as soft surface cards on cream background
  • Add todo as inline dashed row — low visual weight, always accessible
  • Closest to Calm's visual rhythm — most "journal-like"

B — Sidebar + List

Persistent left sidebar for goals, capacity, and navigation. Main area for the todo list. More information-dense; feels more like a structured workspace.

Direction B key traits

  • Persistent sidebar — goals and capacity always in view without scrolling
  • Goal streaks visible at a glance in sidebar — reinforces goal connection
  • More information-dense than A — suits power users and desktop contexts
  • Navigation always accessible — better for multi-section app on web
  • Less "journal," more "workspace" — slightly less warm, more structured

C — Card Stack

Blush pink header with capacity bar, goals strip below, each todo as an elevated card. More visual weight per task. Works well on mobile; feels expressive and warm.

Direction C key traits

  • Pink header creates strong brand moment at the top — feels most distinctively "Tend"
  • Cards give each todo visual weight — good for short lists; can feel heavy with 8+ todos
  • Goals strip is scrollable — compact but always reachable
  • Works especially well on mobile — card pattern is very touch-friendly
  • More expressive and warm than A or D; less structured than B

D — Ultra Minimal

Almost no chrome — just the list. Capacity shown as dots, goals as tiny color dots per row. For users who want their todo list to feel like a clean notebook page.

Direction D key traits

  • Almost invisible chrome — the todos are everything; nothing competes
  • Capacity shown as dots (like a progress indicator) — abstract but readable
  • Goal identity via color dot per row — minimal but scannable
  • Fastest to scan a long list — lowest visual weight per row
  • May feel too sparse for users who need the app to feel "present" and warm
